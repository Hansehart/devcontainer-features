# Dev Container Features

| Feature | Reference | Description |
|---|---|---|
| Claude Code CLI | `ghcr.io/hansehart/devcontainer-features/claude-code` | Installs the Claude Code CLI on a selectable channel. |
| Headless Chrome | `ghcr.io/hansehart/devcontainer-features/headless-chrome` | Installs `chrome-headless-shell` (Chrome for Testing) + runtime libs. |
| LaTeX (TeX Live) | `ghcr.io/hansehart/devcontainer-features/latex` | Installs TeX Live from a pinned snapshot (scheme-basic + collections). |
